# Emotion Detection AI Project (Final Project)

This is a Flask-based web application developed as part of the IBM Final Project.

It uses the Watson NLP Emotion Predict API to analyze text and detect emotions such as anger, disgust, fear, joy, and sadness.

## Features
- Emotion detection from user input text
- Returns emotion scores
- Identifies dominant emotion
- Web interface using Flask
